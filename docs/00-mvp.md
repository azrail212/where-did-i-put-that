## Goal

Save where an item is stored and find it later in seconds.

## MVP Screens

1. Home

- Search bar
- List of items (newest first)

2. Add Item

- Item name (required)
- Location / Where it is (required)
- Save
- Edit
- Delete

## Data (stored locally)

Item:

- id (string)
- name (string)
- location (string)
- createdAt (ISO string)

## MVP Rules

- Add takes < 10 seconds
- Search finds items by name
- Everything works offline

## Out of scope (not in MVP)

- Rooms/boxes
- Photos
- Accounts / cloud sync
- Settings
